FASE 1: FOUNDATION & SETUP (Sprint 0-1)
Prompt 1: Project Structure & CI3 Setup
plain

Buatkan struktur project CodeIgniter 3 lengkap untuk "Sistem Tracer Study Perguruan Tinggi v3.1" dengan HMVC (Modular Extensions).

Requirements:
- PHP 7.4+ compatible
- HMVC structure dengan folder modules: alumni, survey, iku, auth, kurikulum, laporan, stakeholder
- Setup config.php: base_url dinamis, encryption_key random, session_driver = 'database'
- Setup database.php dengan konfigurasi MySQL
- Setup autoload: database, session, form_validation, url, file, security
- Setup routes untuk HMVC
- Setup hooks: AuthHook, AuditHook, MaintenanceHook
- Folder uploads dengan subfolder: avatars, exports, bukti
- .htaccess untuk rewrite dan security headers
- composer.json dengan dependency: phpoffice/phpspreadsheet, dompdf/dompdf, phpunit/phpunit

Jangan buat CRUD dulu, fokus pada struktur folder, konfigurasi, dan base classes (MY_Controller, MY_Model).

Prompt 12: IKU-1 Calculator Engine
plain

Buatkan engine perhitungan IKU-1 lengkap:

Library: IkuCalculator.php
- calculate($kohort_id, $prodi_id) - perhitungan utama IKU-1
- calculateBobot($alumni) - hitung bobot per alumni:
  * Bekerja:
    - Gaji ≥ 1.2 UMP + Masa tunggu ≤ 6 bulan = 1.0
    - Gaji ≥ 1.2 UMP + Masa tunggu 7-12 bulan = 0.8
    - Gaji < 1.2 UMP + Masa tunggu ≤ 6 bulan = 0.8
    - Gaji < 1.2 UMP + Masa tunggu 7-12 bulan = 0.6
  * Wirausaha: 1.0 (omzet ≥ UMP) / 0.8 (omzet < UMP)
  * Lanjut Studi: 0.6
  * Belum Bekerja: 0
- checkMinimumResponse($kohort_id) - cek response rate vs threshold
- applyVVPenalty($score, $vv_rate) - kurangi 20% jika V&V < 80%
- getUMPByProvinsi($provinsi, $tahun) - lookup UMP
- reconcileData($alumni_id) - rekonsiliasi jika bekerja + studi (prioritas bekerja)

Formula:
IKU-1 = (Σ Bobot Responden / Total Responden Memenuhi Minimum) × 100

Controller: Iku.php (CLI + Web)
- calculate($kohort_id, $prodi_id) - trigger perhitungan
- dashboard() - view dashboard IKU
- detail($id) - detail perhitungan IKU
- verify($alumni_id) - verifikasi data alumni

Cron Job:
- php index.php iku calculateAll (daily 02:00 WIB)

Business Rules:
- BR-IKU-001: IKU hanya valid jika response rate ≥ minimum
- BR-IKU-002: Sumber kebenaran gaji_aktual > gaji_range
- BR-IKU-003: V&V rate < 80% = pengurangan skor 20%
- BR-IKU-004: Auto-calc daily 02:00 WIB + real-time saat update
- BR-IKU-005: Export Belmawa hanya admin_pusat_karir/super_admin
